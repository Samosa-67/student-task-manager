# Student Task Manager

## Project Description
Student Task Manager is a web application designed to help students organize and manage their academic tasks in a clear and simple way. The system allows users to add tasks, set due dates and times, track overdue work, and manage completed tasks.

## Features
- Add a new task
- Set due date
- Set due time
- Mark task as completed
- Undo completed task
- Delete task
- Automatically organize tasks into:
  - Pending
  - Due Soon / Overdue
  - Completed
- Save tasks using local storage
- Modern and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub
- Trello
- Clockify
- Draw.io

## Purpose
The purpose of this project is to practice software engineering concepts such as planning, version control, documentation, teamwork, and task management.

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a web browser
4. Start adding and managing tasks
